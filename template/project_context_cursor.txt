# Project: "SoftTech" AI & Dev Services Landing Page

**Objective:** Build a modern, responsive landing page for an AI and software development services company named "SoftTech".

**Workflow:**
- Implement features sequentially, one at a time.
- After completing each feature, create a Git commit using the `emojidev` format, followed by a short description in English.

**Design System:**
- **Typography (from Google Fonts):**
  - **Headings:** Use Space Grotesk, Sora, or Clash Display (in Bold or Semibold).
  - **Body/UI:** Use Inter or Manrope (in Regular or Medium).

- **Color Palette:**
  - **Dark Mode (Default):**
    - Background: `#0F0F0F`
    - Main Text: `#E5E5E5`
    - Primary Accent: `#3F88C5`
    - Secondary Accent: `#B8E986`
    - Emphasis/Errors: `#FF6B6B`
  - **Light Mode (Alternative):**
    - Background: `#F4F6F8`
    - Text: `#2E2E2E`
    - Accents: `#5D5FEF`, `#29B6F6`

**Features to Implement:**
Create modern, responsive sections with original, compelling copy and integrate Tabler icons where appropriate.

1.  **Navbar:**
    - "SoftTech" logo (text + Tabler icon).
    - Menu with anchor links: "About", "Services", "Portfolio", "Stack", "Contact".

2.  **Hero Section:**
    - Animated particle background with blur effects.
    - Write compelling sales copy for AI & software development services.

3.  **About Section.**
4.  **Services Section.**
5.  **Stack Section.**
6.  **Contact Section.**
